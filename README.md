# GRCH: Geometric Residual Curvature Hypothesis

A non-dark-matter explanation of cosmic structure growth.

## Overview
GRCH proposes that dark-matter-like effects arise from persistent spacetime curvature remnants from the early universe.

## Key Features
- Dynamic equation of state w(a) with tanh transition
- Relaxation timescale τ ~ H₀⁻¹
- Clustering factor S(a)
- Field theory: R² + R□⁻¹R action
- CLASS implementation
- Bullet Cluster simulation

## Repository Structure
- `src/` - CLASS modification example
- `simulations/` - Bullet Cluster N-body simulation
- `data/` - DESI fσ8 data
- `paper/` - LaTeX source
